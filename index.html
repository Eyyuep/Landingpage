Wer bin ich?

Mein name ist Eyyüp Baran, ich bin durchlaufe aktuelle eine Umschulung zum Fachinformatiker im Bereich der Anwendungsentwicklung.
Mein erster Berühungspunkt mit der IT war 2013 während eines Praktikum zum Informationstechnischen Assistenten. 2019 habe ich noch ein weiteres Praktikum in der Universitätsklinik in Essen (AÖR) absolviert.
Dort habe ich viel über HTML, CSS & JavaScript erfahren. Aktuell beschäftige ich mich viel mit Java, Python, welches bei meinem Aktuellen Praktikumsbetrieb Gang und Gäbe ist.
Aktuell vertiefe ich meine Java Kenntnisse, speziell mit dem Framework Spring Boot in verbindung mit JavaScript und Datenbanken(PostgresSQL,MySQL,MongoDB...)
Auch mit Progammen wie Jira und Confluence habe ich bereits Erfahrungen gesammelt.
